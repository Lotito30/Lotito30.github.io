de programacion para treabajar con una base de datos relacionales como ORACLE y MYSQL, etc.
MYSQL es un interpretador de MYSQL, es un servidor de base de datos.
MYSQL permite crear base de datos y tablas, insertar datos, modificarlos, eliminarlos, y ordenarlos,
hace consultas y realizar muchas operaciones, etc., resumiendo: administrar base de datos.

Este tutorial tiene por objetivo acercar los conceptos iniciales para introducirse
en el mundo de las bases de datos.

Tipos de datos en MYSQL

Varchar

Se usa para almacenar cadenas de caracteres.Una cadena es una secuencia de caracteres.
se coloca entre 'Hola'.
El tipo "VARCHAR" define una cadena de longitud variable en la cual determinamos el maximo
de caracteres. Para alcenar cadenas hasta 30 caracteres, definimos un campo varchar(30).

Int

Se usa para guardar valores numeros enteros, de -2000000000 a 2000000000 aproximadamente.
definimos campos de este tipo cuando queremos representar, por ejemplo, cantidades.
